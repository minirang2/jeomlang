# 보안 정책 (Security Policy)

이 프로젝트의 보안 취약점을 안전하게 보고하고 협력해 주셔서 감사합니다.

## 지원 버전

현재 보안 업데이트가 적용되는 버전:

- 버전 1.1.0 이상 (가장 최신 버전, 기준: 2026/5/9)

구버전은 보안 패치가 보장되지 않을 수 있습니다.

## 취약점 신고 방법

보안 취약점을 발견한 경우, 공개 이슈로 등록해주시고 discussion에 상세하게 기록해주세요:

- GitHub Issues에 올리거나 비공개로 연락
- 가능한 경우 상세한 재현 방법 포함

## 포함해주면 좋은 정보

- 취약점 설명
- 재현 단계
- 영향 범위
- 가능한 경우 PoC (개념 증명)

## 대응 정책

- 접수 후 가능한 빠르게 확인합니다  
- 필요 시 패치 및 릴리즈를 진행합니다  

## 참고

이 프로젝트는 실험적인 프로젝트로, 예상치 못한 동작이 발생할 수 있습니다.
하지만 보안 관련 문제는 최대한 신중하게 처리합니다.
