# 기여 가이드

이 프로젝트에 관심을 가져주셔서 감사합니다!

## 기여 방법

1. 이 저장소를 Fork 합니다  
2. 새로운 브랜치를 생성합니다:

```bash
git checkout -b feature/your-feature
```

3. 변경 사항을 작성합니다  
4. 정상적으로 동작하는지 테스트합니다  

5. 커밋합니다:

```bash
git commit -m "Add: 기능 설명"
```

6. 자신의 포크 저장소에 푸시합니다:

```bash
git push origin feature/your-feature
```

7. Pull Request를 생성합니다  

## 규칙

- 특별한 이유가 없다면 기존 문법을 깨지 않습니다
- 변경된 내용을 구체적으로 알려줘야합니다.
